# CRM System Documentation

Документация по архитектуре и разработке enterprise CRM‑системы для управления многомиллионными базами телефонных номеров.

## Структура документации

- [architecture_overview.md](architecture_overview.md) — общая архитектура, сущности, роли, интеграции
- [backend_plan_dev1.md](backend_plan_dev1.md) — технический план и структура задач для backend‑разработчика
- [frontend_plan_dev2.md](frontend_plan_dev2.md) — технический план и структура задач для frontend‑разработчика
- [api_contracts.md](api_contracts.md) — описание основных API и схем данных

## Ключевые требования

- Работа с многомиллионными базами номеров
- Open‑source стек (Python/Django REST или Node.js/NestJS + React/Vue + PostgreSQL)
- Локальный запуск с возможностью переноса на сервер/в облако
- SFTP‑обмен с партнерами
- Интеграция с SoftPhone
- Роли: Администратор и Менеджер с разграничением прав
- Стандартный аудит‑лог
- Расширяемость для будущей модернизации

## Для разработчиков

Каждый файл содержит технические планы, этапы разработки и рекомендации по использованию ИИ‑помощников для генерации кода, тестов и документации.
